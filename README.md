
# Project Title

**Balloon Pop Game**

# Description

The Balloon Pop Game is a simple interactive game developed using Java. Balloons move across the game window, and the player must click on them to pop them and earn points. The objective is to pop as many balloons as possible before the time runs out. The game provides an enjoyable experience while demonstrating Java GUI programming and event handling concepts.

# Tools and Technologies
Java
Java Swing (GUI)
AWT Event Handling
Java Timer Class
Object-Oriented Programming (OOP)
Features
Graphical User Interface (GUI)
Random balloon generation
Balloon movement animation
Mouse-click balloon popping
Score tracking system
Countdown timer
Start game button
Restart game option
Game Over screen
Final score display
# Expected Outcome
Develop a fully functional Java-based game.
Gain experience in Java GUI design.
Learn mouse event handling techniques.
Understand timer implementation and animation.
Improve problem-solving and programming skills.
Demonstrate object-oriented programming concepts in a practical project.
# Project Scope
Single-player game.
User interacts through mouse clicks.
Balloons appear and move automatically.
Player earns points by popping balloons.
Game ends when the timer reaches zero.
Final score is displayed to the player.
